# Milestone Execution Template

## Inputs

- Milestone ID
- Approved scope
- Explicit out of scope
- Allowed files
- Verification commands

## Steps

1. Confirm branch and remotes.
2. Read `docs/00_SOURCE_OF_TRUTH.md`.
3. Select the relevant canonical skill.
4. List acceptance criteria.
5. Implement only approved scope.
6. Run verification.
7. Complete final evidence review.
8. Commit and push only the approved branch.

## Output

- Changed files
- Verification results
- Acceptance status
- Evidence links
- Risks and blockers
