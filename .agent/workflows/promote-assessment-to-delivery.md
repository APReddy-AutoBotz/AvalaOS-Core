# Promote Assessment To Delivery

## Trigger
Use when an approved opportunity should become tracked delivery work.

## Inputs
- assessment_id
- approved_recommendation
- target_workspace/project

## Steps
1. Confirm approval and allowed conversion.
2. Create initiative.
3. Create default epics.
4. Create seed task groups.
5. Carry forward assumptions, risks, rationale.
6. Link delivery items back to source assessment.

## Done Criteria
- Delivery objects created
- Traceability link preserved
- No orphaned delivery items
