# Post-Merge Verification Template

## Inputs

- Merged branch
- Merge commit
- Target branch
- Required tag, if approved

## Steps

1. Pull the target branch.
2. Confirm remotes.
3. Run milestone verification commands.
4. Confirm tag policy.
5. Record post-merge evidence.

## Output

- Target branch status
- Verification result
- Tag status
- Follow-up actions
