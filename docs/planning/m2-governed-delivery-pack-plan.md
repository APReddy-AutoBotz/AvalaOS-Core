# M2 Governed Delivery Pack Plan

## Goal

Make Delivery Lite a governed handoff surface that preserves source decisions, approvals, and evidence.

## Scope

Approved work item lineage, owner/status tracking, blockers, evidence links, and handoff audit.

## Explicit Out Of Scope

Full Jira replacement, scoring ownership, runtime automation, and generic project management expansion.

## Files Allowed

Delivery handoff docs, approved Delivery Lite files, tests, and evidence docs.

## Acceptance Criteria

Delivery items retain source assessment, approval state, evidence reference, owner, and status.

## Verification Commands

`npm run typecheck`, `npm run test`, `npm run build`.

## Risks

Scope creep into generalized task management.

## Stop Conditions

Orphaned work items, missing approval lineage, or Delivery modifying scores.
