## Copy And Empty-State Fix Summary

### App Shell

- Header primary action now leads with Avala Assess when Assess is enabled.
- Header subtitle now says `Start with process evaluation`.
- App primary action candidate order now evaluates Process Catalog before Studio.

### Assess

- `Run Decision Engine` was replaced with `Calculate deterministic score`.
- `Generate Docs Pack` was replaced with `Record Studio handoff snapshot`.
- `Create Delivery Backlog` was replaced with `Record Delivery handoff snapshot`.
- Process Detail handoff now says `Open Avala Studio handoff`.
- Handoff readiness preview now states that deterministic scoring records a handoff pack for human review.

### Studio

- Docs empty state now points to Avala Studio and reviewed source context.
- Project selector now describes Avala Studio drafting documents inside a project context.
- Studio form copy now says `Avala Studio Controls`, `Draft provider`, and `Generate governed document pack`.
- Studio generation error alert is now buyer-safe and does not surface raw provider details to the user.

### Delivery

- Generated Artifacts export copy now labels exports as review artifacts.
- Work item import modal now restates lineage and evidence-ref preservation.
- Work item import modal now states that docs-only lineage remains partial and is not Assess-backed evidence.
- Delivery Pack export copy now asks users to review incomplete lineage and missing evidence refs before external sharing.

### Monitor And Deferred Views

- `AI Insights` was relabeled to `Monitor Insights`.
- Disabled Monitor insight state now explains the workspace state instead of showing a generic load failure.
- Reports placeholder is now buyer-safe for the current demo path.

### Setup And Template Library

- Visible `Upgrade to Premium` and `Coming Soon` wording was removed from Organization Setup.
- Template Library now displays `Workspace Pack` and masks legacy premium wording in visible pack descriptions.
- Locked template CTAs now say `Requires workspace configuration`.
- Starter-pack source data was not changed.
