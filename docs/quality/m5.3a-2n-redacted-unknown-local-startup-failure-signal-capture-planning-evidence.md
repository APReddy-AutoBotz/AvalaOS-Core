## Planning Summary

The M5.3a-2n plan defines a future redacted signal-capture mechanism for the `unknown-local-startup-failure` path.

The plan explains:

- Blind retries are no longer useful without safer structured signal metadata.
- Guessed raw-pattern expansion is no longer enough without redacted metadata.
- Future implementation should capture only structured categories.
- Future implementation must never emit raw output or local values.
- Future implementation must fail closed if safe metadata cannot be produced.

## Redacted Signal-Capture Objective

Future M5.3a-2o implementation should produce only approved signal fields and approved category values:

- Startup attempt status.
- Exit result class.
- Command phase class.
- Output size bucket.
- Line count bucket.
- Timeout flag.
- Pattern family count totals.
- Sanitized category candidates.
- Confidence category.
- Output safety result.
- Scratch cleanup result.
- Safety block reason category.
- No-secrets, no-local-path, and no-raw-log confirmation categories.

## Allowed Future Signal Fields

- `startupAttemptStatus`: `attempted`, `not-attempted`
- `exitResultClass`: `success`, `nonzero-exit`, `timeout`, `spawn-error`, `killed`, `unknown`
- `commandPhaseClass`: `preflight`, `supabase-cli-invocation`, `image-resolution`, `image-pull`, `container-create`, `service-start`, `health-wait`, `cleanup`, `unknown`
- `outputSizeBucket`: `none`, `small`, `medium`, `large`, `oversized`, `unknown`
- `lineCountBucket`: `none`, `small`, `medium`, `large`, `oversized`, `unknown`
- `timeoutFlag`: boolean
- `knownPatternFamilyCounts`: approved family labels mapped to counts only
- `sanitizedCategoryCandidates`: category labels only, no matched text
- `confidence`: `high`, `medium`, `low`
- `outputSafetyResult`: `passed`, `failed`
- `scratchCleanupResult`: `removed`, `not-created`, `failed`
- `safetyBlockReasonCategory`: `none`, `raw-output-unsafe`, `secret-like`, `local-path-like`, `target-like`, `container-image-id-like`, `raw-log-like`, `output-too-large`, `unknown`
- no-secrets confirmation category
- no-local-path confirmation category
- no-raw-log confirmation category
