## Root Cause

M4.0a critical finding `M4A-C03` found that Docs-to-Delivery import created Delivery work items without preserving source lineage or evidence refs. M4.1b gave generated artifacts an Assess source context, but the Delivery import path still did not copy that source context onto imported tasks, and the handoff ledger used generated work item titles as `evidenceRefs` instead of evidence reference ids.

## Before And After

Before:

- Generated work item import created Delivery epics/tasks without task-level lineage.
- Delivery adapter did not persist `task.sourceLineage`.
- Handoff ledger event evidence refs were generated work item titles.
- Delivery Pack could consume lineage if present, but the import path did not populate it.

After:

- Imported Story/Task records receive compact `TaskSourceLineageMetadata`.
- Task metadata persistence round-trips `sourceLineage`.
- Handoff ledger evidence refs use upstream evidence ref ids when present.
- Handoff metadata records flat lineage counts and source ids.
- Docs-only imports remain visible as partial or missing lineage; no fake Assess source is created.
- Generated artifacts from a valid Assess handoff retain their own `sourceContext`.

## Lineage Model Evidence

The pure helper `buildDocsToDeliveryLineage` builds compact task lineage from the generated artifacts, document generation id, generated work item, timestamp, and handoff ledger entry id.

Fields carried when available:

- Immediate source: `sourceModule`, `documentGenerationId`, `sourceGenerationId`, `sourceArtifactKey`, `sourceArtifactTitle`, `sourceWorkItemTitle`.
- Upstream Assess source: `upstreamSourceModule`, `processId`, `assessmentId`, `sourceContextLabel`, `sourceDecisionPackRef`, `sourceHandoffPackRef`.
- Evidence: `evidenceRefs`, `assumptionRefs`, `handoffLedgerEntryIds`.
- Gap visibility: `lineageCompleteness`, `lineageNotes`, `sourceStatus`.

Completeness behavior:

- Complete when generation id, Assess process/assessment ids, and evidence ref ids exist.
- Partial when source context or generation id exists but evidence/source ids are incomplete.
- Missing when neither source context nor generation id exists.
