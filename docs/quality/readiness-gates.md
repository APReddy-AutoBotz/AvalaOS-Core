# Readiness Gates

Reference: `docs/00_SOURCE_OF_TRUTH.md`.

| Gate | Name | Required Evidence |
| --- | --- | --- |
| Gate 0 | Migration complete | Main normalized, migration tag, no old repo push. |
| Gate 1 | Build Control Pack complete | M0.2 docs, skills, inventory, wording scan, verification. |
| Gate 2 | Govern Lite hardening complete | Registry fields, approval/evidence rules, audit status. |
| Gate 3 | Governed Delivery Pack complete | Handoff lineage, owners, blockers, source evidence. |
| Gate 4 | Server-side AI/BYOK hardening complete | Server-side execution, key references, usage audit. |
| Gate 5 | Demo readiness complete | Demo story, verified flow, claim-safe copy. |
| Gate 6 | Health proof pack complete | Separate Health proof story, no core-platform merge. |
| Gate 7 | Pilot readiness | Verified controls, runbooks, evidence, open-risk list. |

No gate passes without evidence and verification.
