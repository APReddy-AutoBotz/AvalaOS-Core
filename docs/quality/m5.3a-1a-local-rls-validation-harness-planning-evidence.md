## Remote-Target Rejection Design

Future runner requirements:

- Fail if the DB target is not explicitly local.
- Treat local as `localhost`, `127.0.0.1`, Docker local network, or another AP-approved local-only target.
- Reject hosted Supabase URLs and personal hosted project values.
- Never log the DB target value.
- Never silently accept `DATABASE_URL`.
- Never fall back to ignored local secret files.

M5.3a-1a did not create or change `supabase/config.toml`.
