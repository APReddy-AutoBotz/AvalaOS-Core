## Implementation Summary

M5.3a-2o strengthened the existing local startup failure classifier so it emits a redacted structured signal after the single approved local startup attempt. The classifier interface remains unchanged and accepts only `--classify` and `--help`.

The classifier still records raw startup output only inside generated scratch while deriving safe metadata. It does not print raw output, stack traces, target values, host values, port values, DB URLs, local paths, IDs, auth headers, claim values, or row payloads. Generated scratch was removed before final scans.
