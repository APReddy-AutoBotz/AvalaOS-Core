## Per-Table SELECT Policy Summary

`assess_processes`:

- Requires `org_id IS NOT NULL`.
- Requires `workspace_id IS NOT NULL`.
- Requires `deleted_at IS NULL`.
- Requires `public.is_active_workspace_member(workspace_id, org_id)`.

`assessments`:

- Requires `org_id IS NOT NULL`.
- Requires `workspace_id IS NOT NULL`.
- Requires `deleted_at IS NULL`.
- Requires `public.is_active_workspace_member(workspace_id, org_id)`.
- Requires `process_id` to resolve to a same-organization, same-workspace, non-deleted `assess_processes` row.

`assessment_review_events`:

- Requires `org_id IS NOT NULL`.
- Requires `workspace_id IS NOT NULL`.
- Requires `public.is_active_workspace_member(workspace_id, org_id)`.
- Requires `assessment_id` to resolve to a same-organization, same-workspace, non-deleted `assessments` row.
- Requires `process_id` to resolve to a same-organization, same-workspace, non-deleted `assess_processes` row.
- Has no row-level `deleted_at` predicate because `assessment_review_events` has no `deleted_at` column.

`projects`:

- Requires `org_id IS NOT NULL`.
- Requires `workspace_id IS NOT NULL`.
- Requires `deleted_at IS NULL`.
- Requires `public.is_active_workspace_member(workspace_id, org_id)`.
- Requires populated `source_process_id` to resolve to a same-organization, same-workspace, non-deleted `assess_processes` row.
- Requires populated `source_assessment_id` to resolve to a same-organization, same-workspace, non-deleted `assessments` row.

`document_generations`:

- Requires `org_id IS NOT NULL`.
- Requires `workspace_id IS NOT NULL`.
- Requires `project_id IS NOT NULL`.
- Requires `deleted_at IS NULL`.
- Requires `public.is_active_workspace_member(workspace_id, org_id)`.
- Requires `project_id` to resolve to a same-organization, same-workspace, non-deleted `projects` row.
- Requires populated `source_process_id` to resolve to a same-organization, same-workspace, non-deleted `assess_processes` row.
- Requires populated `source_assessment_id` to resolve to a same-organization, same-workspace, non-deleted `assessments` row.

`delivery_work_items`:

- Requires `org_id IS NOT NULL`.
- Requires `workspace_id IS NOT NULL`.
- Requires `project_id IS NOT NULL`.
- Requires `deleted_at IS NULL`.
- Requires `public.is_active_workspace_member(workspace_id, org_id)`.
- Requires `project_id` to resolve to a same-organization, same-workspace, non-deleted `projects` row.
- Requires populated `document_generation_id` to resolve to a same-organization, same-workspace, non-deleted `document_generations` row.
- Requires populated `source_process_id` to resolve to a same-organization, same-workspace, non-deleted `assess_processes` row.
- Requires populated `source_assessment_id` to resolve to a same-organization, same-workspace, non-deleted `assessments` row.

## Required Policy Confirmations

- All six artifact policies require `org_id IS NOT NULL`.
- All six artifact policies require `workspace_id IS NOT NULL`.
- All six artifact policies use `public.is_active_workspace_member(workspace_id, org_id)`.
- No artifact policy uses `public.is_active_org_member(org_id)`.
- `assess_processes`, `assessments`, `projects`, `document_generations`, and `delivery_work_items` require `deleted_at IS NULL`.
- `assessment_review_events` has no row-level `deleted_at` predicate because the table has no `deleted_at` column.
- Parent/source reference checks explicitly require same organization and same workspace where populated.
- Source references do not grant visibility by themselves.
- JSON fields such as `metadata`, `artifacts`, `payload`, `evidence_items`, `assumptions`, `review`, `scores`, and `source_lineage` are not authorization inputs.
- Archived and workflow/status fields do not independently block SELECT in this slice.
