## Root Cause

The scored Process Detail rendering path used stale flat Decision Pack fields:

- `decisionPack.whyThisRecommendation`
- `decisionPack.governanceControls`

Those fields are not part of the current `DecisionPack` type. The generated deterministic score record stores the recommendation rationale under `decisionPack.recommendedOperatingModel.whyThis` and stores control groups under `decisionPack.governance.*Controls`. The separate `HandoffPack` stores a flattened `governanceControls` list for downstream handoff use.

## Contract Mismatch Summary

| Rendering need | Stale field | Current source |
| --- | --- | --- |
| Decision rationale | `decisionPack.whyThisRecommendation` | `decisionPack.recommendedOperatingModel.whyThis` |
| Governance controls | `decisionPack.governanceControls` | `decisionPack.governance.auditControls`, `dataControls`, `monitoringControls`, `securityControls`, `modelProviderControls` |
| Handoff governance fallback | N/A | `handoffPack.governanceControls` |
| Required docs | Direct optional chain to array map | Guarded `handoffPack.requiredDocumentTypes` normalization |
| Backlog seeds | Direct optional chain to array slice/map | Guarded `handoffPack.suggestedBacklogItems` normalization |
