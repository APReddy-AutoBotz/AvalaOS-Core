## Implementation Scope

M5.3a-1b implements a local-only harness skeleton and scenario contract. It does not implement helper functions, RLS policies, tenant policy SQL, schema migrations, durable fixture data, seed data, backfill, constraint validation, runtime adapters, UI, routing, view keys, Sidebar behavior, demo data, scoring, provider execution, export behavior, deployment config, CI workflows, Reports/Teams/Admin behavior, or the separate Health repository.

The harness is infrastructure only. It does not prove tenant isolation and does not make hosted runtime ready.

## Local-Target Rejection Behavior

The JS runner requires an explicit local target token before any future DB action can proceed. Accepted local target tokens are limited to:

- `localhost`
- `127.0.0.1`
- `loopback`
- `docker`
- `docker-local`

The runner rejects missing targets, unknown arguments, unsafe target token shapes, and environment-derived target variables when no explicit local target is accepted. It does not print target values.

## Fixture And Scenario Contract

The implemented contract declares these negative scenarios:

- anonymous denied
- authenticated non-member denied
- disabled organization member denied
- disabled workspace member denied
- Org A user cannot read Org B organization metadata
- Org A user cannot read Org B membership metadata
- Workspace A user cannot read Workspace B workspace membership metadata
- Workspace A user cannot read Workspace B scoped role metadata
- system role visibility remains denied unless AP approves it
- service-role behavior remains server-only and not browser-callable

The implemented contract declares these positive scenarios:

- own active profile readable
- own active organization membership readable
- own active workspace membership readable
- organization metadata readable only through active organization membership
- workspace metadata readable only through active workspace membership
- role metadata scoped to active membership

No local fixture rows are inserted by this milestone.
