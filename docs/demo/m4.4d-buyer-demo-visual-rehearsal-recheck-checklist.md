## Local Run Instructions

Use the existing repository-supported local app path:

1. Check out the M4.4d branch.
2. Install dependencies only if the workspace is missing `node_modules`.
3. Start the local app:

```powershell
npm run dev -- --host 127.0.0.1 --port 4199
```

4. Open `http://127.0.0.1:4199/`.
5. Use a clean browser profile, private window, or Playwright session with cleared cookies, local storage, and session storage.
6. Set the desktop viewport to approximately `1440x1000`.
7. Keep DevTools console open, or use Playwright CLI console output, and record errors separately from expected local-dev warnings.
