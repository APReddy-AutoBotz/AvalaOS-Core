# Audit Event Model

Reference: `docs/00_SOURCE_OF_TRUTH.md`.

## Event Fields

- Event ID
- Timestamp
- Actor
- Source object
- Action
- Before and after summary
- Decision or approval reference
- Evidence reference
- Risk level
- Result

## Event Types

- Assessment scored
- Recommendation generated
- Document seeded
- Governance card reviewed
- Approval recorded
- Delivery handoff created
- Action blocked
- Incident or kill switch triggered

Audit proves what happened. It does not replace approval.
