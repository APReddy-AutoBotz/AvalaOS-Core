# Evidence Requirements Model

Reference: `docs/00_SOURCE_OF_TRUTH.md`.

| Context | Evidence Required |
| --- | --- |
| Assessment score | Source inputs, formula version, deterministic output. |
| Recommendation | Score rationale, gates, rejected alternatives. |
| Document generation | Source assessment, missing inputs, reviewer. |
| Governance card | Owner, autonomy, risk, allowed and blocked actions. |
| Approval | Approver, decision, timestamp, evidence reference. |
| Delivery handoff | Source assessment, document link, work item lineage. |

Missing evidence is a finding, not a reason to invent evidence.
