# M4.0a Logic And Data Flow Map

## Golden Path Logic Map

Canonical path:

Organization -> Process Catalog -> Guided Assessment -> Deterministic Score -> Recommendation / Decision Pack -> Avala Govern Lite -> Avala Studio Handoff -> Generated Documents -> Work Items -> Avala Delivery Lite Board -> Avala Monitor Dashboard.

## View Coverage Findings

| Comparison | Result | Risk |
| --- | --- | --- |
| `View` enum vs `VIEW_MODULE_MAP` | All enum values are mapped except no separate Admin key; `WORKSPACE` maps to Docs only. | Admin is not a first-class module in `ProductModuleKey`. |
| `View` enum vs App render | Top-level App handles global Assess views, Docs Forge/Workspace/Templates/Docs, Dashboard/Portfolio, and delegates remaining delivery views. | Delegation is broad but permission is not centrally enforced at render time. |
| Sidebar vs App render | Sidebar exposes fewer views than App can render. | User/localStorage/defaultView can reach views not normally exposed. |
| Sidebar permission checks vs direct route state | Sidebar hides unauthorized links, but `handleViewChange` only checks module enablement, not permissions. | RBAC visibility and actual route access can diverge. |
| Module enablement vs org scope | Non-org views are blocked if module disabled; org scope bypasses normal view gating and renders `OrganizationSetupView`. | Admin route semantics are overloaded. |
| Hidden/disabled modules vs flow access | `handlePrimaryAction` and `handleAssessToDocsHandoff` check module enablement for Docs in some places; some persisted view state can still redirect. | Mostly safe but needs central route guard for enterprise. |

## Enterprise Demo Interpretation

The golden path can be shown conditionally as a controlled demo if the narrator is precise:

- Show deterministic Assess scoring as the source of decision authority.
- Show Govern Lite as a derived governance review surface.
- Show Studio generation as draft, editable, review-required content.
- Show Delivery Lite and Delivery Pack as governed handoff surfaces.
- Show Monitor dashboard/portfolio as demo signals, not validated enterprise reporting.

Do not claim:

- Live provider path is validated.
- Provider governance Admin UI is complete.
- Imported work items preserve complete lineage today.
- Browser/local-demo AI is acceptable for pilot or production.
- Reports or full enterprise analytics are complete.
- Compliance certification exists.
